Instalation

Requirement

  1. Unity 4.6
  2. Android SDK
  3. Java Development Kit
This template is using Unity 4.6.9 and I recommend to using the same version. If you using Unity 5 above, it still worked but I don't guaranted about performance. I got really bad performance on Unity 5 above.
Make sure you have set all SDK and Java path to unity.

 

Instalation

  1. Open and Import Crossy Way project to Unity
  2. After compiling and importing assets complete, Go to File > Built Settings , choose Android and click Switch Platform
  3. Go to Edit > Project Settings > Player and add 'SOFT PROCESS' to Scripting Define Simbols. (Without Quote)
  4. Done :)

FAQ

Share Not Attach Image?

This because write permission, change it to external. Go to Edit > Project Settings > Player, then open Other and find Write Access and change to external.

Google Play Game Service not working?

If you are just started developing Android Game, usually missing some step. Guide GPGS what I create is started from you have creating Google Play Game Service and just implemented to Crossy Way Game Template. This is just simple reminder 

  • Make sure your SHA1 and package name is correct at Linked Apps.
  • Have set to alpha or beta on Testing Page
  • Have adding your gmail account on Testing Page.
  • Have sign your release keystore.
  • If you sure all was correct but GPGS not login completely, Go to Unity and open Assets/Plugins/Android/MainLibProj/AndroidManifest.xml and check if your Google Play Service ID was correct

Unity Ads not working?

If you are using Unity 5.2 above, and follow My guide. Yeay you are right. It will not work because Unity Ads Implementation has change since Unity 5.2. But my script UnityAdController.cs has optimized for Unity Ads and its worked. Just follow this and add Scripting Define Simbols (see at Instalation Section) 'unityad' without quote
